# Lumina Calculator

A stunning, glassmorphic calculator application built with React, TypeScript, and Tailwind CSS.

## Features

- **Modern Design**: Glassmorphism UI with light/dark mode support.
- **Responsive**: Works beautifully on mobile and desktop.
- **History Tape**: View and restore past calculations.
- **Keyboard Support**: Full keyboard navigation for rapid entry.
- **Scientific Notation**: Automatically handles large numbers.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- clsx & tailwind-merge

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm start
   ```

## Usage

- Use the on-screen keypad or your physical keyboard.
- Press the **History** icon to see past calculations.
- Press the **Sun/Moon** icon to toggle themes.
